关于我们

这是一个使用 Flask 构建的 Web 应用。

技术栈

  • 后端：Flask + SQLAlchemy + Flask-JWT-Extended + Flask-SocketIO
  • 前端：Jinja2 模板 + 原生 JavaScript + Canvas API + video.js
  • 数据库：SQLite
  • 实时通信：WebSocket（弹幕）
  • 图标：Font Awesome

功能

  • 用户注册与登录（JWT + Session）
  • 视频上传与播放（支持 20GB 大文件）
  • 弹幕系统（滚动/顶部/底部三种模式，实时 WebSocket）
  • 头像上传与选择
  • 个人中心（视频管理 + 密码修改）
  • 联系我们留言
