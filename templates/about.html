关于视频社区

一个类 B 站风格的视频分享与弹幕互动平台

技术栈

Flask SQLAlchemy JWT SocketIO SQLite Canvas API video.js pytest

主要功能

  • 用户注册与登录（JWT + Session）
  • 视频上传与播放（支持 20GB 大文件）
  • 实时弹幕系统（8 色可选，WebSocket）
  • 点赞 / 投币 / 收藏 / 分享
  • 关注取关 + 动态推送
  • 个性化推荐引擎
  • 用户空间 + 动态时间线
  • 头像上传与 MIME 校验

安全特性

  • CSRF 防护（Flask-WTF）
  • JWT 过期自动跳转登录
  • 图片 MIME 类型校验（PIL）
  • 密码哈希存储（Werkzeug）
  • 文件名安全处理

感谢使用视频社区，祝你玩得开心！
