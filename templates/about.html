关于我们

一个类 B 站风格的视频社区网站。

技术栈

  • 后端：Flask + SQLAlchemy + Flask-JWT-Extended + Flask-SocketIO
  • 前端：Jinja2 模板 + 原生 JavaScript + Canvas API + video.js
  • 数据库：SQLite + Flask-Migrate
  • 实时通信：WebSocket（弹幕）
  • 测试：pytest
  • 图标：Font Awesome

功能

  • 用户注册与登录（JWT + Session 双认证）
  • 视频上传与播放（支持 20GB 大文件）
  • 弹幕系统（滚动/顶部/底部，8 色可选，实时 WebSocket）
  • 视频互动（点赞、投币、收藏、分享）
  • 关注/取关 + 动态推送
  • 用户空间（投稿列表 + 动态时间线）
  • 头像上传与图片库选择
  • 个人中心（视频管理 + 密码修改）
  • 联系我们留言
